### What's New in v1.0.4

This release brings several major UI/UX polishing details and crucial form validations to ensure a flawless game setup!

- **Keyboard Avoidance UI**: Forms no longer get obscured by the soft keyboard. Text inputs float smoothly over the keyboard when typing.
- **Neo-brutalist Aesthetic Tweaks**: 
  - Player Cards now feature a perfectly sized highlight-badge around names and a sleek separator divider.
  - Mortgage property cards now dynamically pull the exact background color of their property group, paired with bold black uppercase typography for maximum contrast.
  - "PAY OFF" buttons are now a sleek neutral cream to prevent clashing with the dynamic backgrounds.
  - Refined the base hex values for Brown, Dark Blue, and Station Grey properties for enhanced legibility without losing their classic Monopoly identity.
- **Unique Setup Validation**: The game setup screen now rigorously checks for duplicate player names and throws an in-line bright red Neo-brutalist error box if a collision is detected (preventing start).
- **Sound Polish**: The fanfare "finish" sound effect is now correctly isolated to when a game actually ends, and gracefully falls back to a simple button tap sound when just viewing past stats from the Home Screen.

*Includes the standard testable Android `.apk` built straight from the latest `main` branch.*
